# bam-mut-analyzer 使用指南

## 1. 功能介绍

bam-mut-analyzer 是一个用于分析BAM文件中突变情况的工具，能够详细统计和分析DNA合成过程中的突变类型和频率。

### 主要功能

- **突变检测**：检测并分类替换、插入、缺失等突变类型
- **细分类分析**：对突变进行细分类，如Del1、Del2、Del3等缺失类型
- **统计报告**：生成多种统计报告，包括突变频谱、长度分布等
- **批量处理**：支持批量处理多个BAM文件
- **参考序列支持**：从Excel文件读取参考序列和样本信息

## 2. 安装方法

### 前提条件
- Go 1.16 或更高版本
- 相关依赖库

### 编译步骤

```bash
# 进入项目目录
cd /path/to/SynthesisAnalyzer

# 编译bam-mut-analyzer
go build -o bin/bam-mut-analyzer cmd/bam-mut-analyzer/main.go

# 或直接运行
go run cmd/bam-mut-analyzer/main.go [参数]
```

## 3. 使用方法

### 基本用法

```bash
# 基本用法
bam-mut-analyzer -d <输入目录> -o <输出目录>

# 使用Excel文件
bam-mut-analyzer -i <Excel文件> -d <输入目录> -o <输出目录>

# 自定义参数
bam-mut-analyzer -i <Excel文件> -d <输入目录> -o <输出目录> -head 27 -tail 20 -max-sub 5 -n 4
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-d` | 输入目录，包含样本子目录 | 必需 |
| `-o` | 输出目录 | 输入目录/mutation_stats |
| `-i` | Excel文件，包含样本顺序和参考序列 | 可选 |
| `-head` | 头切除长度 | 27 |
| `-tail` | 尾切除长度 | 20 |
| `-max-sub` | 最大替换个数阈值 | 5 |
| `-n` | N-mer 统计的 N 值 | 4 |
| `-log-level` | 日志级别 (debug, info, warn, error) | info |

## 4. 输入文件格式

### Excel文件格式

Excel文件需要包含以下列：

| 列名 | 说明 |
|------|------|
| 样品名称 | 样本的名称 |
| 靶标序列 | 靶标部分的序列 |
| 合成序列 | 合成部分的序列 |
| 后靶标 | 后靶标部分的序列 |

**示例Excel文件内容**：

| 样品名称 | 靶标序列 | 合成序列 | 后靶标 |
|----------|----------|----------|--------|
| Sample1  | ATGC...  | CGTA...  | GCAT...|
| Sample2  | ATGC...  | CGTA...  | GCAT...|

### BAM文件要求

- BAM文件需要包含MD标签，用于突变检测
- 建议提供对应的BAI索引文件
- BAM文件应该按照样本组织在子目录中

## 5. 输出文件说明

### 主要输出文件

1. **total_mutation_stats.csv**：总突变统计
   - 包含所有突变类型的统计信息
   - 显示每种突变的数量、频率等

2. **read_type_by_sample.csv**：样本read类型统计
   - 按样本统计不同类型的read数量
   - 包括匹配、插入、缺失、替换等类型

3. **read_type_summary.csv**：汇总read类型统计
   - 汇总所有样本的read类型统计
   - 包含各种突变类型的百分比

4. **read_subtype_stats.csv**：细分类统计
   - 详细统计各种细分类的突变情况
   - 包括Del1、Del2、Del3、Dup1等子类型

5. **mutation_spectrum.csv**：突变频谱
   - 显示不同突变类型的分布
   - 帮助分析突变模式

6. **insert_length_distribution.csv**：插入长度分布
   - 统计不同长度插入的分布情况

7. **delete_length_distribution.csv**：缺失长度分布
   - 统计不同长度缺失的分布情况

8. **deletion_position_heatmap.csv**：缺失位置热力图数据
   - 用于生成缺失位置的热力图
   - 显示不同位置的缺失频率

## 6. 突变分类说明

### 主要突变类型

1. **替换（Substitution）**：单个碱基的替换
2. **插入（Insertion）**：碱基的插入
3. **缺失（Deletion）**：碱基的缺失

### 细分类

#### 缺失细分类
- **Del1**：长度为1的缺失
- **Del2**：长度为2的缺失
- **Del3**：长度大于2的缺失

#### 插入细分类
- **Dup1**：长度为1，与相邻碱基一致的插入
- **Dup2**：长度为2，与相邻碱基一致的插入
- **Ins1**：长度为1，非重复的插入
- **Ins2**：长度为2，非重复的插入
- **Ins3**：长度大于2的插入

#### 替换细分类
- **DupDel**：与前一个碱基相同的替换
- **DelDup**：与后一个碱基相同的替换
- **MismatchA>G**：A→G的替换
- **MismatchC>T**：C→T的替换
- **OtherMismatch**：其他类型的替换

## 7. 性能优化

### 已实现的优化

1. **循环现代化**：使用range over int替代传统for循环
2. **并发处理**：使用goroutine并发处理多个BAM文件
3. **内存优化**：合理使用内存，避免内存泄漏
4. **I/O优化**：减少I/O操作，提高处理速度
5. **字符串处理优化**：改进MD字符串解析，使用直接切片替代strings.TrimPrefix

### 性能建议

- 对于大量样本，建议使用多核CPU以提高处理速度
- 确保有足够的内存用于处理大型BAM文件
- 对于特别大的BAM文件，可以考虑增加max-sub参数值

## 8. 故障排除

### 常见错误及解决方法

1. **Excel文件读取失败**
   - 错误信息：`未找到'样品名称'列`
   - 解决方法：确保Excel文件包含所有必需的列

2. **BAM文件分析失败**
   - 错误信息：`assignment to entry in nil map`
   - 解决方法：确保BAM文件格式正确，包含必要的标签

3. **内存不足**
   - 错误信息：`out of memory`
   - 解决方法：减少并发处理的文件数量，或增加系统内存

4. **输出文件空白**
   - 问题：某些统计文件内容空白
   - 解决方法：检查BAM文件是否包含足够的reads，或调整参数设置

## 9. 示例分析流程

### 步骤1：准备输入文件

1. 准备BAM文件，按照样本组织在子目录中
2. 创建Excel文件，包含样本信息和参考序列

### 步骤2：运行分析

```bash
# 运行bam-mut-analyzer
bam-mut-analyzer -i samples.xlsx -d ./bam_files -o ./results

# 查看分析进度
# 分析完成后会显示总体统计信息
```

### 步骤3：查看结果

```bash
# 查看输出文件
ls ./results/

# 查看总突变统计
cat ./results/total_mutation_stats.csv

# 查看样本read类型统计
cat ./results/read_type_by_sample.csv
```

### 步骤4：分析结果

1. **总突变统计**：了解整体突变情况
2. **read类型统计**：分析不同类型read的分布
3. **细分类统计**：深入了解突变的具体类型
4. **长度分布**：分析插入和缺失的长度分布
5. **位置分布**：分析突变在序列中的位置分布

## 10. 高级功能

### 自定义参考序列

通过Excel文件，可以为每个样本指定不同的参考序列，实现更准确的突变分析。

### 批量处理

工具支持批量处理多个BAM文件，自动识别输入目录中的所有BAM文件并进行分析。

### 自定义切除长度

根据不同的实验设计，可以通过`-head`和`-tail`参数自定义头尾部切除长度。

## 11. 代码结构

### 主要文件

- **main.go**：主入口，处理命令行参数
- **pkg/stats/**：核心统计功能包
  - **Sample.go**：样本信息管理
  - **SampleStats.go**：样本统计信息
  - **MutationStats.go**：突变统计信息
  - **sam.go**：SAM/BAM文件解析
  - **writer.go**：输出文件写入逻辑
  - **struct.go**：数据结构定义
  - **type.go**：类型定义
  - **util.go**：工具函数

### 核心流程

1. 读取命令行参数
2. 读取Excel文件（如果提供）
3. 查找BAM文件
4. 并发处理BAM文件
5. 生成统计文件
6. 输出结果

## 12. 注意事项
